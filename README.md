# Job Tracker

A full-stack job application tracking platform built with Next.js, TypeScript, PostgreSQL, and Prisma.

## Overview

Job Tracker helps users organize and manage their job search by tracking applications, monitoring interview progress, and viewing application analytics in a centralized dashboard.

## Planned Features

* User authentication
* Create, update, and delete job applications
* Application status tracking
* Dashboard and analytics
* Interview notes
* Resume management
* AI-powered resume feedback
* Job match scoring

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Next.js Server Actions / Route Handlers
* Prisma ORM

### Database

* PostgreSQL (Neon)

### Authentication

* Auth.js

### Deployment

* Vercel

## Getting Started

Install dependencies:

npm install

Run the development server:

npm run dev

Open http://localhost:3000 to view the application.
